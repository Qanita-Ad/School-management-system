# School Management System in C++

A simple **console-based School Management System** in C++ that allows administrators to **manage students, teachers, and classes**. All data is stored locally in text files.  

---

## Features

- **Student Management**  
  - Add new students with details like name, roll number, class, and contact info.  
  - View student information.  
  - Update or delete student records.  

- **Teacher Management**  
  - Add new teachers with details like name, ID, subjects, and contact info.  
  - View teacher information.  
  - Update or delete teacher records.  

- **Class Management**  
  - Assign students to classes.  
  - Assign teachers to subjects and classes.  

- **File-Based Storage**  
  - All data (students, teachers, classes) is stored in `.txt` files.  
  - Easy to retrieve, update, and delete records.  

- **Input Validation**  
  - Handles invalid menu options and incorrect input gracefully.
